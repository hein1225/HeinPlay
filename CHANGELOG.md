# 更新日志

## 1.0.3

### 新增

- **M3U8 本地去广告**
  - 参考 TVBox 广告过滤思路，在客户端本地完成 M3U8 广告过滤，不再依赖 LunaTV 服务端广告配置。
  - 新增 `M3u8AdFilter`：支持少数 URL 过滤、不连续片段组分析、SCTE-35/CUE 广告标记识别、URL/域名特征匹配及安全回退机制。
  - 新增 `LocalM3u8Proxy`：本地 HTTP 代理服务，统一补全请求头（Referer、Origin、Range 等），兼容 ExoPlayer / media_kit / video_player 三种播放器后端。
  - 播放器后端自动识别本地代理地址，避免对 `127.0.0.1` 进行二次 LunaTV 代理包装。
  - 设置中保留去广告开关，默认关闭，开启后对所有 M3U8 播放器生效。

- **分类页默认排序调整**
  - 电影、电视剧、综艺、动漫分类中，除「近期热门」外，所有小分类（含「全部」）默认排序改为「首映时间」。
  - 切换小分类时自动重置为默认排序；「近期热门」仍保持排序维度隐藏。

### 修复

- **播放失败自动切换源**
  - 设置页将「自动切换源」开关与等待时间选项合并为同一卡片，关闭开关时隐藏等待时间选项。
  - 修复等待时间设置未真正生效的问题：现在以设置时长为总上限，等待播放器报告有效时长后才视为成功，超时则触发切换源。

- **继续播放与播放记录**
  - 海报下方不再显示年份，改为显示该条记录对应的播放源名称。

- **二维码登录**
  - 修复退出登录后再次使用二维码登录时提示「发送失败」的问题；`RemoteInputService` 单例生命周期内保持 StreamController 可用。

- **更新弹窗**
  - 弹窗尺寸加大，完整显示更新标题与日志内容。
  - 下载进度条固定在底部始终可见，不再随日志滚动到下方。

- **关于页**
  - 开源仓库地址改为国内 GitCode 地址 `https://gitcode.com/gcw_QbmhmbO8/HeinPlay`。

### 变更

- 应用版本号统一更新至 `1.0.3`（`pubspec.yaml`、关于页、`UpdateService`、`LunaTV User-Agent`）。

## 1.0.2

### 修复

- **收藏夹实时同步**
  - 修复详情页添加/取消收藏后“我的”页面未实时刷新的问题。
  - 收藏状态变更时同步写入本地 `SharedPreferences`，并通过 `ProfileRefreshNotifier` 通知“我的”页面立即重新加载收藏列表，无需重启应用。

- **跳过片头片尾设置**
  - 修复跳过配置对话框中焦点无法移动到“片头”“片尾”按钮及时间输入框的问题。
  - 修复点击“添加”后焦点未继续下移、停留在添加按钮上的问题。
  - 电视剧类型跳过配置改为自动识别并直接确认，无需用户手动输入。

- **分类页海报墙焦点**
  - 修复分类页海报网格长按方向键时焦点异常跳回“筛选与排序”的问题。
  - 现在方向键同时支持 `KeyDownEvent` 与 `KeyRepeatEvent`，避免长按遥控器导致重复焦点遍历。
  - 海报网格行内循环：行尾按右进入下一行首个海报，行首按左进入上一行末尾海报；仅首个海报按左返回筛选/排序栏。

### 新增 / 优化

- **“我的”页面弹窗重构**
  - 播放记录与收藏夹弹窗改为屏幕 92%×88% 的居中 Dialog，展示更多内容。
  - 新增工具栏：支持“批量选择”“清空”功能。
  - 批量模式显示“完成 / 删除(数量) / 清空”，选中海报显示勾选标记与标题高亮。
  - 删除与清空操作二次确认，删除后同步清理本地存储并调用 LunaTV 接口同步服务器。
  - 弹窗内焦点导航完善：工具栏按钮可聚焦，下键进入首张海报，首行海报上键返回工具栏，海报网格使用几何位置导航。

- **版本号**
  - 应用版本号统一更新至 `1.0.2`（`pubspec.yaml`、关于页、`UpdateService`、`LunaTV User-Agent`）。

## 1.0.1

### 修复

- **全屏播放遥控器确认键**
  - 修复控制栏隐藏时按确认键无法播放/暂停的问题。
  - 新增 `HardwareKeyboard` 全局按键兜底处理，确保平台视图夺焦时遥控器按键仍可正常响应。
  - 控制栏隐藏时，左右键继续支持快进/快退，下键显示控制栏并聚焦功能键。

- **全屏播放焦点顺序**
  - 修复按下键显示控制栏后焦点不直观的问题，现优先聚焦“跳过”按钮；跳过不可用时回退到播放/暂停按钮。

- **分类页焦点导航**
  - 修复电影、电视剧、综艺、动漫分类页中，焦点位于子分类或“筛选与排序”时按上键错误跳到其他顶部导航的问题，现正确返回当前分类导航项。
  - 修复筛选/排序面板按返回键时误触发应用退出确认对话框的问题，现优先关闭面板并返回分类页。

- **播放源与网络请求**
  - 播放请求自动补充 `Referer` 与 `Origin` 头，提升源站兼容性。
  - 尊重 `VideoDetail.proxyMode` 字段，对需要代理的 M3U8/HLS 源自动启用代理。
  - 修复手动切换播放源时集数被重置为第 1 集的问题，现保留当前集数。
  - 修复部分本地 `file://` M3U8 文件被错误走 asset 加载的问题。
  - 优化 LunaTV 搜索超时与重试机制，提升源搜索速度。

- **跳过片头片尾与广告**
  - 支持读取 LunaTV `/api/skipconfigs` 中的跳过片段（片头、片尾、广告）。
  - 跳过配置对话框新增“广告”类型，广告片段默认自动跳过。
  - M3U8 广告过滤默认关闭，可在播放器设置中手动开启。

### 新增

- **应用内更新双渠道**
  - 默认使用国内渠道（GitCode）检查更新。
  - “我的”页面手动检查更新时支持切换国内/GitHub 渠道。
  - 下载 APK 前先请求 `REQUEST_INSTALL_PACKAGES` 权限，再通过系统安装器安装。

- **全屏播放触摸控制（平板/触屏设备）**
  - 单击显示/隐藏控制栏。
  - 双击播放/暂停。
  - 长按左右区域 3 倍速快进/快退。
  - 左侧上下滑调节亮度，右侧上下滑调节音量。
  - 左右滑调节播放进度。

- **播放体验优化**
  - 全屏播放时启用屏幕常亮，退出播放后自动释放。
  - 播放器后端初始化失败时输出更详细的日志，便于排查问题。

### 变更

- 升级 `permission_handler` 到 `^12.0.3`（需 Android `compileSdkVersion 35`，当前 Flutter 版本默认提供 36）。
- 通过 `flutter pub upgrade` 更新 `wakelock_plus`、`path_provider`、`video_player_android` 等 13 个依赖到最新兼容版本。

## 1.0.0

### 新增功能

- **首页与分类浏览**
  - 首页智能推荐，聚合热门影视内容。
  - 电影、电视剧、综艺、动漫四大分类，支持小分类横向切换。
  - 分类页支持「筛选与排序」面板，可按类型、地区、年份、特色及排序方式筛选。

- **影片详情与选集**
  - 查看影片详情、评分、简介、演职员信息。
  - 多播放源切换，剧集选集一键播放。

- **全屏播放体验**
  - 支持播放/暂停、快进/快退、进度拖动。
  - 画面比例切换，适配不同视频源。
  - 播放器控制栏自动隐藏，支持 TV 遥控器方向键与确认键操作。

- **多播放器后端**
  - 集成 ExoPlayer、MediaKit、video_player 三种播放器后端，可根据场景自由切换。

- **跳过片头片尾**
  - 支持按电视剧维度设置跳过片段，自动跳过片头片尾并触发下一集。

- **搜索**
  - 关键词搜索影视内容。
  - 最近搜索记录与搜索推荐，提升查找效率。

- **个人中心**
  - 播放历史、收藏管理。
  - 扫码登录。
  - 设置入口与 M3U8 代理配置。

- **TV 遥控优化**
  - 完整的遥控器焦点导航与方向键控制。
  - 顶部导航栏焦点循环，分类页焦点层级管理。
  - 筛选面板返回键优先关闭面板，避免误退出应用。
  - 分类页按上键回到当前所在分类导航项。

- **应用内更新**
  - 检测 GitHub Releases 新版本。
  - 点击「立即更新」先请求安装未知应用权限，再分步完成 APK 下载与安装。
