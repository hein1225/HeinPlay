# 更新日志

## 1.0.0

### 新增功能

- **首页与分类浏览**
  - 首页智能推荐，聚合热门影视内容。
  - 电影、电视剧、综艺、动漫四大分类，支持小分类横向切换。
  - 分类页支持「筛选与排序」面板，可按类型、地区、年份、特色及排序方式筛选。

- **影片详情与选集**
  - 查看影片详情、评分、简介、演职员信息。
  - 多播放源切换，剧集选集一键播放。

- **全屏播放体验**
  - 支持播放/暂停、快进/快退、进度拖动。
  - 画面比例切换，适配不同视频源。
  - 播放器控制栏自动隐藏，支持 TV 遥控器方向键与确认键操作。

- **多播放器后端**
  - 集成 ExoPlayer、MediaKit、video_player 三种播放器后端，可根据场景自由切换。

- **跳过片头片尾**
  - 支持按电视剧维度设置跳过片段，自动跳过片头片尾并触发下一集。

- **搜索**
  - 关键词搜索影视内容。
  - 最近搜索记录与搜索推荐，提升查找效率。

- **个人中心**
  - 播放历史、收藏管理。
  - 扫码登录。
  - 设置入口与 M3U8 代理配置。

- **TV 遥控优化**
  - 完整的遥控器焦点导航与方向键控制。
  - 顶部导航栏焦点循环，分类页焦点层级管理。
  - 筛选面板返回键优先关闭面板，避免误退出应用。
  - 分类页按上键回到当前所在分类导航项。

- **应用内更新**
  - 检测 GitHub Releases 新版本。
  - 点击「立即更新」先请求安装未知应用权限，再分步完成 APK 下载与安装。

### 支持平台

- Android TV（主要目标平台）
- Web
- Windows

### 其他

- 适配 Android TV LEANBACK_LAUNCHER。
- Release 构建使用 `hain_tv_keystore.jks` 签名。
